# Computer Architecture - Lecture 1 (Introduction)

## Overview

This repository contains the introductory lecture on **Computer Architecture and Computer Organization**. 
It explains the functional design of computer systems, hardware implementation, instruction sets, memory, 
and the basic architecture of a computer.

---

## Learning Objectives

After completing this lecture, students will be able to:

- Define Computer Architecture and Computer Organization.
- Differentiate between architecture and organization.
- Understand instruction sets, data types, and addressing modes.
- Explain the role of control signals, data paths, and memory technology.
- Identify the basic components of a computer system.

---

## Computer Architecture

Computer Architecture refers to the **functional design and behavior** of a computer system. It describes **what a computer does** and how programmers interact with it.

### Key Characteristics

- Focuses on **what the computer does**
- Describes the **functional behavior** of the system
- Visible to the programmer
- Includes:
  - Instruction Sets (ISA)
  - Data Types
  - Addressing Modes

### Example

Different processors, such as Intel and AMD, may implement the same instruction set architecture (ISA) while having different internal hardware organizations.

---

## Computer Organization

Computer Organization focuses on the **physical implementation** of a computer system. It explains **how the computer performs its tasks** using hardware components.

### Key Characteristics

- Focuses on **how the computer works**
- Describes hardware implementation details
- Not visible to the programmer
- Includes:
  - Control Signals
  - Data Paths
  - Memory Technology
  - Hardware Interconnections

---

## Architecture vs Organization

| Computer Architecture | Computer Organization |
|----------------------|----------------------|
| What the computer does | How the computer does it |
| Functional behavior | Physical implementation |
| Visible to the programmer | Not visible to the programmer |
| Instruction sets and data types | Control signals and memory technology |

> **Architecture = What**  
> **Organization = How**

---

## Basic Architecture of a Computer

A computer system consists of three major components:

### 1. Input Devices

Input devices are used to provide data and instructions to the computer.

Examples:

- Keyboard
- Mouse
- Scanner
- Joystick

---

### 2. Central Processing Unit (CPU)

The CPU is the brain of the computer and consists of the following units:

#### Control Unit (CU)

The Control Unit manages and coordinates all operations within the computer system by fetching, decoding, and executing instructions.

#### Arithmetic Logic Unit (ALU)

The ALU performs:

- Arithmetic Operations (+, -, ×, ÷)
- Logical Operations (AND, OR, NOT, Comparison)

#### Memory Unit

The memory unit stores data, instructions, and intermediate results required during program execution.

---

### 3. Output Devices

Output devices present processed information to the user.

Examples:

- Monitor
- Printer
- Speaker
- Earphones

---

## Conclusion

Computer Architecture defines the functionality and capabilities of a computer system, 
whereas Computer Organization explains the implementation of those capabilities through hardware components. 
Understanding both concepts provides a strong foundation for studying computer systems and processor design.
